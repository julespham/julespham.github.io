Maxymiser

What is Maxymiser?

Website utilized for A/B testing (split testing).

Note: "A/B testing is when you split your audience to tests a number of variations of a campaign and determines which performs better."

Okay...

How is this beneficial to us (The Hartford)?

  1. Make any UI changes(small to big) on any website at the Hartford
  2. Changes other projects won't affect your variant
  3. Utilizes campaign montioring
  4. QA tool for testing is offered

Let's Get Technical, Technical

  1. Scripts are written in JavaScript
  2. Could attach action events such as button clicks, and page views
  3. Create multiple variants so you could have different changes on the same page
  4. Drag and drop UI
There's only one caveat
Variants are also written in JQuery.
