Maxymiser

What is Maxymiser?

Website utilized for A/B testing (split testing). Note: "A/B testing is when you split your audience to tests a number of variations of a campaign and determines which performs better."

Okay...

How is This beneficial to us (The Hartford)?

  1. We could make any UI changes(small to big) on any link at the Hartford
  2. Changes from other UI in other projects won't affect the variant
  3. Great for analyzing data to determine which UI works best for customers

Let's Get Technical, Technical

  1. Scripts are written in JavaScript
  2. Could attach action events such as button clicks, and page views
  3. Create multiple variants so you could have different changes on the same page
  4. Drag and drop UI
There's only one caveat
Variants are also written in JQuery.
